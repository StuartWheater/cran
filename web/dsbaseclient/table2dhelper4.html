table2dhelper4 {dsbaseclient}R Documentation

Finalizes the output of the function table2d.ds

Description

This is an INTERNAL i.e. it is not 'exported' and hence not visible to users. This function is only called by the client function ds.table2d. It calls other internal functions namely table2dhelper1, table2dhelper2 and table2dhelper3.

Usage

  table2dhelper4(server.func.output, var.name.1,
    var.name.2)

Arguments

server.func.output

a list object returned by the server side function table2d.ds

var.name.1

a character string, the name of the first input numerical vector

var.name.2

a character string, the name of the second input numerical vector

Value

a list that contains the elements returned by the client function ds.table2d

Author(s)

Burton, P.; Gaye, A.
