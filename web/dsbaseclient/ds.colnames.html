ds.colnames {dsbaseclient}R Documentation

Retrieves column names of a matrix-like object

Description

this function is similar to R function colnames

Usage

ds.colnames(datasources = NULL, x = NULL)

Arguments

datasources

a list of opal object(s) obtained after login in to opal servers; these objects hold also the data assign to R, as dataframe, from opal datasources.

x

a matrix-like object

Value

a string containing column names of the given object

Author(s)

Gaye, A.; Isaeva, J.

Examples

{

# load that contains the login details
data(logindata)

# login
opals <- datashield.login(logins=logindata,assign=TRUE)

# Example 1: Get the column names of the assigned datasets
ds.colnames(datasources=opals, x=quote(D))

# Example 2: Vectors are wrong input to the colnames function
## Not run:  ds.colnames(datasources=opals, x=quote(D$LAB_TSC))
}
