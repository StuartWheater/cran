ds.quantilemean {dsbaseclient}R Documentation

Compute the quantiles

Description

This function calculate the mean and quatile values of a quantitative variable without the minimum and maximum which are revealing.

Usage

ds.quantilemean(datasources = NULL, xvect = NULL, type = "combine")

Arguments

datasources

a list of opal object(s) obtained after login in to opal servers; these objects hold also the data assign to R, as dataframe, from opal datasources.

xvect

a numeric vector.

type

a character which represent the type of graph to display. If type is set to 'combine' pooled values are displayed and sumamries a returned for each study if type is set to 'split'

Value

one or more summaries depending the argument type

Author(s)

Gaye, A.

Examples

{

# load that contains the login details
data(logindata)

# login and assign specific variable(s)
myvar <- list("LAB_HDL")
opals <- datashield.login(logins=logindata,assign=TRUE,variables=myvar)

# Example 1: plot a combined histogram of the variable 'LAB_HDL' - default behaviour
ds.quantilemean(datasources=opals, xvect=quote(D$LAB_HDL))

# Example 2: Plot the histograms separately (one per study)
ds.quantilemean(datasources=opals, xvect=quote(D$LAB_HDL), type="split")

}
