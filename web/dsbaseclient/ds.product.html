ds.product {dsbaseclient}R Documentation

Computes a product

Usage

ds.product(datasources = NULL, xlist = NULL, newobj = NULL)

Arguments

datasources

a list of opal object(s) obtained after login in to opal servers; these objects hold also the data assign to R, as dataframe, from opal datasources.

xlist

a list of numerical to compute a product for.

newobj

the name of the product object. If this argument is set to NULL, the name of the new object is 'prodOutput'.

Details

The function computes the product of up to five elements

Value

a message is displayed when the action is completed.

Author(s)

Gaye, A.

Examples

{

# load the file that contains the login details
data(logindata)

# login and assign the required variables to R
myvar <- list("LAB_TSC","LAB_HDL")
opals <- datashield.login(logins=logindata,assign=TRUE,variables=myvar)

# compute the product of 'LAB_TSC' by 'LAB_HDL' and assign it to 'P'
prodinput <- list(quote(D$LAB_TSC), quote(D$LAB_HDL))
ds.product(datasources=opals, xlist=prodinput, newobj=NULL)
}
