table1dhelper4 {dsbaseclient}R Documentation

Finalizes the output of the function table1d.ds

Description

this is an INTERNAL function required by the client function ds.table1d.

Usage

  table1dhelper4(input)

Arguments

input

a list object returned by the server side function table1d.ds

Details

the function is not 'exported' and hence not visible to user and is only called by ds.table1d

Value

a list that contains the elements returned by the client function ds.table1d

Author(s)

Burton, P.; Gaye, A.
