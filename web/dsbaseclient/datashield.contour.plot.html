datashield.contour.plot {dsbaseclient}R Documentation

Creates a contour plot

Description

Generates a contour plot of the given data values.

Usage

  datashield.contour.plot(opals, xvect, yvect)

Arguments

opals

character strings that represent the URL of the servers where the study datasets are stored.

xvect

a numerical vector

yvect

a numerical vector

Value

a contour plot.

Author(s)

Burton, P.
