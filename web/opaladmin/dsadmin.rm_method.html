dsadmin.rm_method {opaladmin}R Documentation

Remove Datashield Method

Description

Remove a Datashield method from Opal(s).

Usage

  dsadmin.rm_method(opal, name, type = "aggregate")

Arguments

opal

Opal object or list of opal objects.

name

Name of the method, as it is accessed by Datashield users.

type

Type of the method: "aggregate" (default) or "assign"
