Get Datashield Methods

Description

Get Datashield methods from Opal(s).

Usage

  dsadmin.get_methods(opal, type = "aggregate")

Arguments

opal

Opal object or list of opal objects.

type

Type of the method: "aggregate" (default) or "assign"
