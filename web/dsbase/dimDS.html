dimDS {dsbase}R Documentation

Returns the dimensions of a table structure

Description

Returns the number of rows and the number of columns.

Usage

dimDS(x)

Arguments

x

a character, the name of a matrix or a dataframe.

Details

This is similar to R base function dim but here the input is a character.

Value

the number of rows and the number of columns

Author(s)

Gaye, A.
