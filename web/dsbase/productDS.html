productDS {dsbase}R Documentation

Computes a product

Description

calculates the product of two or more objects.

Usage

productDS(xlist)

Arguments

xlist

a list of numerical to compute a product for.

Value

a numerical

Author(s)

Gaye, A.

Examples

## Not run: 
# load the file that contains the login details
library(opal)
data(logindata)

# login and assign the required variables to R
myvar <- list("LAB_TSC","LAB_HDL")
opals <- datashield.login(logins=logindata,assign=TRUE,variables=myvar)

# compute the of 'LAB_TSC' by 'LAB_HDL' and assign it to 'P'

datashield.assign(opals, "P", cally)
