subsetvar.ds {dsbase}R Documentation

Derives a valid binary variable from a continuous variable

Description

Derives a valid binary variable from a continuous variable

Usage

  subsetvar.ds(xvect = NULL, operator = NULL,
    threshold = NULL)

Arguments

xvect

a continuous numeric variable

operator

an integer that sets the operator 1 for '>', 2 for '<', 3 for '>=', 4 for '<=', 5 for '==' and 6 for '!='

threshold

a numeric value that sets the threshold

Details

This function runs only if the set threshold (i.e. >=) is met or not met by 1 single observation.

Value

a vector where the entries that met the condition are assigned the value 1 or 0 otherwise

Author(s)

Gaye, A.

Examples

## Not run: 
# load the login data
data(logindata)

# login and assign some variables to R
myvar <- list("PM_BMI_CONTINUOUS")
opals <- datashield.login(logins=logindata,assign=TRUE,variables=myvar)

# run the function with for the condition '>=30'
var <- quote(D$PM_BMI_CONTINUOUS)
operator <- 3
threshold <- 30
call.object <-  call("subsetvar.ds", var, as.numeric(operator), as.numeric(30))
datashield.assign(opals, "bmi30", call.object)

## End(Not run)
