isValid.ds {dsbase}R Documentation

Checks if an input is valid

Description

This function checks if an input variable is valid. A numeric variable is not valid if it has > 0 and < 5 observations and a factor variable is not valid if any of its levels (classes) has a count of between 1 and 4.

Usage

isValid.ds(xvect)

Arguments

xvect

a numeric or factor vector.

Value

a boolean TRUE if input is valid or stops the process otherwise

Author(s)

Gaye, A.

Examples
