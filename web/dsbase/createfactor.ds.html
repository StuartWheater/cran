createfactor.ds {dsbase}R Documentation

Generates a factor variable

Description

The functions uses the input factor and generates a new factor with new levels.

Usage

createfactor.ds(xvect = NULL, categories = NULL)

Arguments

xvect

a factor vector

categories

the levels of the factor to generate

Value

a factor vector that has the levels of the original vector and some additional levels

Author(s)

Gaye, A.

Examples

## Not run: 
# load the login data
library(opal)
data(logindata)

# login and assign specific variable(s)
myvar <- list("PM_BMI_CATEGORICAL")
opals <- datashield.login(logins=logindata,assign=TRUE,variables=myvar)

# get the levels of variable PM_BMI_CATEGORICAL in study1
current.levels <- datashield.aggregate(opals[1], quote(levels(D$PM_BMI_CATEGORICAL)))

#' generate a new factor vector which has an additonal level '4'
call.object <- call("createfactor.ds", quote(D$PM_BMI_CATEGORICAL), as.list(current.levels[[1]], "4"))
datashield.assign(opals, "bmi_f", call.object)

 
## End(Not run)
